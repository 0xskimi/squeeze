# Squeeze

Make your videos smaller, right in your browser. Nothing gets uploaded.

**Live:** [squeeze.tentacore.xyz](https://squeeze.tentacore.xyz)

A side project by [Tentacore](https://tentacore.xyz). Inspired by [compress.lochie.me](https://compress.lochie.me/).

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
